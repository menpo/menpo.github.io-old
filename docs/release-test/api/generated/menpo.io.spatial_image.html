class menpo.io.spatial_image.BNTImporter(filepath)[source]¶

Bases: menpo.io.spatial_image.SpatialImageImporter

Allows importing the BNT file format from the bosphorus dataset. This reads in the 5 channels (3D coordinates and texture coordinates), splits them appropriately and then triangulates the x and y coordinates to create a surface. The texture path is also given in the file format.

The file format specifies a ‘bad_value’ which is used to denote inaccurate data points. This value is replaced with np.nan. The mesh is created with only those values that are not NaN. The depth image contains all values.

Parameters:

filepath : string

Absolute filepath of the mesh.
