import_builtin_asset¶

menpo.io.base.import_builtin_asset(asset_name)[source]¶

Single builtin asset (mesh or image) importer.

Imports the relevant builtin asset from the ./data directory that ships with Menpo.

Parameters:

asset_name : String

The filename of a builtin asset (see ls_ls_builtin_assets() for allowed values)
Returns:

asset :

An instantiated asset (mesh, trimesh, or image)
