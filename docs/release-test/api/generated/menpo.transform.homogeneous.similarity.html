as_vector()[source]¶

Return the parameters of the transform as a 1D array. These parameters are parametrised as deltas from the identity warp. The parameters are output in the order [a, b, tx, ty], given that a = k cos(theta) - 1 and b = k sin(theta) where k is a uniform scale and theta is a clockwise rotation in radians.

2D

parameter definition
a a = k cos(theta) - 1
b b = k sin(theta)
tx Translation in x
ty Translation in y

Note

Only 2D transforms are currently supported.

Returns:

params : (P,) ndarray

The values that parameterise the transform.
Raises:

DimensionalityError, NotImplementedError :

If the transform is not 2D

jacobian(points)[source]¶

Computes the Jacobian of the transform w.r.t the parameters.

The Jacobian generated (for 2D) is of the form:

x -y 1 0
y  x 0 1

This maintains a parameter order of:

W(x;p) = [1 + a  -b   ] [x] + tx
         [b      1 + a] [y] + ty
Parameters:

points : (N, D) ndarray

The points to calculate the jacobian over
