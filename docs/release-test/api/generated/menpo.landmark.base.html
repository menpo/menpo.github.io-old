class menpo.landmark.base.Landmarkable[source]¶

Bases: object

Abstract interface for object that can have landmarks attached to them. Landmarkable objects have a public dictionary of landmarks which are managed by a menpo.landmark.base.LandmarkManager. This means that different sets of landmarks can be attached to the same object. Landmarks can be N-dimensional and are expected to be some subclass of menpo.shape.pointcloud.Pointcloud. These landmarks are wrapped inside a LandmarkGroup object that performs useful tasks like label filtering and viewing.

class menpo.landmark.base.LandmarkGroup(target, group_label, pointcloud, labels_to_masks)[source]¶

Bases: menpo.visualize.base.Viewable

An immutable object that holds a PointCloud (or a subclass) and stores labels for each point. These labels are defined via masks on the pointcloud. For this reason, the pointcloud is considered to be immutable.

Parameters:
