Module: basis.rbf¶

3 Classes¶

class menpo.basis.rbf.BasisFunction(c)[source]¶

Bases: object

An abstract base class for Basis functions. In the case, radial basis functions. They provide two methods, apply(), which calculates the basis itself, and jacobian_points(), which calculates the derivative of the basis wrt the coordinate system.

Parameters:

c : (L, D) ndarray

The set of centers that make the basis. Usually represents a set of source landmarks.
__init__(c)[source]¶
apply(x)[source]¶

Calculate the basis function on the given residuals. The input is the set of points the basis should be calculated for. The euclidean distance between x and the centers, c, will be used as the residual.

Note

Divisions by zero are avoided and any zero residuals remain zero.

Parameters:

x : (N, D) ndarray

Set of points to apply the basis to.
Returns:

u : (N, L) ndarray

The basis function applied to each distance, \(\lVert x - c \rVert\).
jacobian_points(x)[source]¶

Calculate the derivative of the basis function wrt the coordinate system.

Note

Divisions by zero are avoided and any zero residuals remain zero.

Parameters:

x : (N, D) ndarray

Set of points to apply the basis to.
Returns:

dudx : (N, L, D) ndarray

Tensor representing the first order partial derivative of each points with respect to the centers, over each dimension.
class menpo.basis.rbf.R2LogR2(c)[source]¶

The \(r^2 \log{r^2}\) basis function.

The derivative of this function is \(2 r (\log{r^2} + 1)\).

Note

\(r = \lVert x - c \rVert\)

jacobian_points(x)[source]¶

Apply the derivative of the basis function wrt the coordinate system. This is applied over each dimension of the input vector, x.

Note

..math:

2 (x - c)^T (\log{r^2_{x, l}} + 1) ===
2 (x - c)^T (2 \log{r_{x, l}} + 1)

where:

:math:`r_{x, l} = lVert x - c

Vert``
