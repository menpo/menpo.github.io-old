class menpo.shape.mesh.base.TriMesh(points, trilist=None)[source]¶

Bases: menpo.shape.pointcloud.PointCloud

A pointcloud with a connectivity defined by a triangle list. These are designed to be explicitly 2D or 3D.

Parameters:

points : (N, D) ndarray

The set coordinates for the mesh.

trilist : (M, 3) ndarray, optional

The triangle list. If None is provided, a Delaunay triangulation of the points will be used instead.

Default: None
