R2LogR2RBF¶

class menpo.transform.rbf.R2LogR2RBF(c)[source]¶

Bases: RadialBasisFunction

The \(r^2 \log{r^2}\) basis function.

The derivative of this function is \(2 r (\log{r^2} + 1)\).

Note

\(r = \lVert x - c \rVert\)

Parameters:

c : (L, D) ndarray

The set of centers that make the basis. Usually represents a set of source landmarks.

jacobian_points(x)[source]¶

Apply the derivative of the basis function wrt the coordinate system. This is applied over each dimension of the input vector, x.

Note

..math:

2 (x - c)^T (\log{r^2_{x, l}} + 1) ===
2 (x - c)^T (2 \log{r_{x, l}} + 1)

where:

\(r_{x, l} = \lVert x - c \rVert\)

Parameters:

x : (N, D) ndarray

Set of points to apply the basis to.
Returns:

dudx : (N, L, D) ndarray

The jacobian tensor representing the first order partial derivative of each point wrt the coordinate system
